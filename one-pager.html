Perseus & Mimir: Open-Source Infrastructure
for AI Control & Interpretability

Technical capability brief · Thomas Connally · tcconnally@gmail.com · github.com/Perseus-Computing-LLC
Two production-deployed, MIT-licensed systems that directly address the AI Forge Critical AI Challenges. Perseus (v1.0.8) is a live context engine that resolves environment state and injects it before every LLM interaction — anchoring agents to ground truth. Mimir (v1.0.1) is a persistent memory MCP server with 30 tools, structured entities, journaled decision trails, and hybrid FTS5+vector search. Both are self-contained binaries with no cloud dependencies, deployable on air-gapped infrastructure. Together they give operators end-to-end traceability from context input through memory formation to agent action.

Perseus · Live Context Engine

AI ControlChallenge 2Challenge 4

Pre-resolves environment state at every session boundary and injects a canonical snapshot before the model acts. Eliminates context decay — the dominant failure mode in long-running agentic AI. Listed on the MCP Registry via Smithery.

→ Challenge 4 (Agentic Interpretability): Context snapshots produce verifiable decision-point records. Any agent action is traceable to the exact state that informed it.
→ Challenge 2 (Long-Horizon Failures): Snapshot timeline enables replay and root-cause isolation. Investigators identify when drift entered and which decisions it corrupted.
Version: 1.0.8 · License: MIT · Install: pip install perseus-ctx · MCP: Smithery-listed · perseus.observer

Mimir · Auditable Memory Engine

AI InterpretabilityChallenge 2Challenge 4

30 MCP tools. Rust binary. SQLite + FTS5 + hybrid vector search. Structured entity model with confidence decay, journaled append-only event log, and state management with TTL. Fully local — no API keys, no network calls at runtime.

→ Challenge 2 (Verifiable Accountability): Journaled audit trail transforms black-box failure analysis into forensic investigation. Trace what the system knew, when it learned it, and when contamination entered.
→ Challenge 4 (Agentic Interpretability): 30 typed tools expose the agent's complete mental model. Every entity creation, relation, and insight is timestamped and attributable.
Version: 1.0.1 · License: MIT · Tools: 30 MCP tools · Backend: SQLite+FTS5, no cloud deps · github.com/Perseus-Computing-LLC/mimir
